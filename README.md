 # Student Management System

A Flask-based Student Management System with login, signup, CRUD operations, search, dashboard analytics, and flash messages.

## 🚀 Live Demo

https://student-management-system-4-55ii.onrender.com

## 📂 GitHub Repository

https://github.com/shraa25/student-management-system

## Features

* Login / Signup
* Add Students
* Edit Students
* Delete Students
* Search Students
* Dashboard Analytics
* Flash Notifications
* SQLite Database

## Tech Stack

* Python
* Flask
* SQLite
* Bootstrap
* Jinja2
* Chart.js
* Render
* Git & GitHub
